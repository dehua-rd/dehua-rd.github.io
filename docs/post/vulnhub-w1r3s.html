靶机扫描

sudo nmap -sT -p- --min-rate 1000 $ip -oA tcpScan
sudo nmap -sU -p- --min-rate 10000 $ip -oA udpScan

tcp和udp可以扫描两遍以防止失误

没路子了，再研究研究这个漏洞，好像可以利用。但是看不到输出，使用php的伪协议输出试一下，还是没有，也不报错。
